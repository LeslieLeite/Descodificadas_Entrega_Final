O que é Scratch?

O Scratch é uma linguagem de programação visual que permite criar jogos, animações e diversas histórias interativas. Criada pelo grupo Lifelong Kindergarten da universidade americana MIT, o programa tem como objetivo ensinar a lógica da programação para crianças e adolescentes. Sua dinâmica de funcionamento é bem simples e intuitiva, uma vez que o ele trabalha com blocos para montar, tipo LEGO. Se você não tem noção nenhuma de lógica de programação, então o Scratch é uma excelente ferramenta para começar a criar jogos.

Com o Scratch, é possível criar histórias, jogos e animações com scripts feitos com blocos. É um tipo de programação “visual”, mais simplificada. Isso permite que qualquer um, mesmo sem saber nada sobre programação, consiga criar projetos digitais com mais facilidade.

Logo, é possível programar suas próprias histórias, jogos e animações interativas, compartilhando suas criações com outras pessoas na comunidade online. O Scratch ajuda assim os jovens a aprender a pensar de forma criativa, raciocinar sistematicamente e trabalhar de forma colaborativa - habilidades essenciais para a vida no século 21. Scratch é projetado, desenvolvido e moderado pela Scratch Foundation, uma organização sem fins lucrativos. Além disso, ele é fornecido gratuitamente, basta ter acesso a internet para começar a criar os seus próprios jogos.
